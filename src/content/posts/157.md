准备：
1. 树莓派已经安装系统 （以ubuntu为例）
2. 用ssh客户端连接树莓派

安装：
1.更新并升级
```
apt update && apt upgrade
```
2.安装docker和compose
```
apt install docker && apt install docker-compose
```
3.更换国内镜像源
请查看：https://www.easybhu.cn/archives/156.html

运行：(进入compose项目内运行)
```
docker-compose up
```
