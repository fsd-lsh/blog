期望将如下对象截取前3对key-value
```
var obj = {
  apple: 124,
  banana: 23,
  melon: 19,
  pear: 100,
  orange: 68
};
```

函数
```
  intercept(ob,start,end) {
      let Index_0 = Object.entries(obj).slice(start,end);
      let Index_1 = Object.fromEntries(Index_0);
      return Index_1;
  }
```

调用
```
intercept(obj, 0, 3)
```
