#### 进入容器
```
docker exec -it xxxxxxxxx bash

```

#### 创建临时目录
```
mkdir /var/lib/mysql/temp 
```

#### 在data目录生成新的证书（共8个）
```
cd /usr/bin && ./mysql_ssl_rsa_setup --datadir=/var/lib/mysql/temp --verbose
```
生成后会发现temp目录下有8个新证书

#### 退出容器 并 关闭容器
```
exit
docker stop xxxxxxxxx
```

#### 进入mysql挂载的data目录（以/data/mysql为例）
将`/data/mysql/temp`下全部证书替换到`/data/mysql`下

### 启动容器
```
docker start xxxxxxxxx
```

### ssl所需相关证书替换完成
Done!
