在html中设置锚点定位我知道的有几种方法，在此和大家分享一下：

1、使用id定位：
```
<a href="#1F">锚点1</a> 
<div id="1F"> 
<p> 
11111111111 
</br> 
11111111111 
</br>11111111111 
</br>11111111111 
</br>11111111111 
</br>11111111111 
</br>11111111111 
</br>11111111111 
</br>11111111111 
</br>11111111111 
</br>11111111111 
</br>11111111111 
</br>11111111111 
</br>11111111111 
</br>11111111111 
</br>11111111111 
</br> 
</p> 
</div> 
```
这样的定位可以针对任何标签来定位。

2、使用name定位：
```
<a href="#5F">锚点5</a> 
</br></br></br></br></br></br></br></br></br></br></br></br></br></br></br></br></br></br></br> 
<a name="5F">1111111</href> 
```
使用name属性只能针对a标签来定位，而对div等其他标签就不能起到定位作用。

3、使用js定位：
```
<li class="" onclick="javascript:document.getElementById('here').scrollIntoView()"></li> 
```
