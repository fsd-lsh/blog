最近想基于Docker部署Jenkins服务， 发现重启宿主机后已经配置的Jenkins数据全部丢失

官方文档启动说明比较隐晦，不利于小白理解

这里举例子解决此问题

#### 1.安装好Docker Jenkins镜像
这里不在赘述pull过程
![1.png][1]

#### 2.启动命令
这里的`-v`属性是重点， 挂载容器`/var/jenkins_home`目录到宿主机`/data/jenkins`目录。 
当然你可以把`/data/jenkins`宿主机，目录设置到任何合适的地方。

下面命令如果直接向复制用，请更替`9f9894008293`为您实际的Jenkins容器ID
```shell
docker run  -u root --rm  -d -p 8080:8080 -p 50000:50000 /data/jenkins:/var/jenkins_home 9f9894008293
```
