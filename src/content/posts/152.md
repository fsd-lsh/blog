```shell
curl https://download.argon40.com/argon1.sh | bash
```

安装结束之后，桌面会出现两个快捷方式，“configure”和“uninstall”，用来更改设置和删除驱动。

但也有命令行做这样的行动。风扇的默认设置是：
```
CPU温度	风扇转速
55度	10%
60度	55%
65度	100%
```

要修改设置，键入下面的命令：
```
argonone-config
```

要删除驱动程序，键入下面的命令：
```
argonone-uninstall
```

电源键的功能
```
树莓派状态	按钮动作	功能
关机	短按	开机
开机	长按三秒以上	软关机或切断电源
开机	短按三秒以内	无任何功能
开机	双击	重启
开机	长按五秒以上	强制关机
```
