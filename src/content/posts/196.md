安装依赖

```
npm i iptoolbox
```

项目中引入`iptoolbox`包，
例如希望检测服务器地址：`127.0.0.1`，端口：`9002`
```
import {IP, Network} from "iptoolbox";
const ip = new IP("127.0.0.1");
const state = await ip.portOpen(9002);
console.log(state)
```

链接：https://www.npmjs.com/package/iptoolbox
