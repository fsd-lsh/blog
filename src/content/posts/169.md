### 先开启general_log
```mysql
set global general_log='ON';
```

### 查看general_log_file path
```mysql
show variables like 'general_log%';

#显示类似log位置
#/usr/local/var/mysql/forcedembp.log
```

### 打印
```shell
cat /usr/local/var/mysql/forcedembp.log
```
