在less中使用calc()计算元素宽度
```less
  #lic-notice {
    width: calc(100% - 200px);
    height: auto;
    z-index: 999;
  }
```

样式被编译后宽度错误显示`width: calc(-100%);`
```
#lic-notice[data-v-49507455] {
    width: calc(-100%);
    height: auto;
    z-index: 999;
}
```

修正兼容less写法`~"xx - xx"`
```less
  #lic-notice {
    width: calc(~"100% - 200px");
    height: auto;
    z-index: 999;
  }
```

ok
