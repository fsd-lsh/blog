uniapp微信小程序DOM元素v-show失效

问题代码
```html
<div class="go-pc" v-show="goPcInfo.closable">
  <span>{{ goPcInfo.info }}</span>
</div>
```
```less
<style lang="less" scoped>
#app {
  .editArea {
    .go-pc {
      display: flex;
      background: #e3f2ff;
      padding: 32rpx 36rpx;
      position: relative;
      ......
```

出现这个问题原因是因为`.go-pc`元素为`flex`，`v-show`本质为`none`、`block`，优先级比`flex`低,
所以解决这个问题只需要将`v-show`修改为`v-if`
```html
<div class="go-pc" v-if="goPcInfo.closable">
  <span>{{ goPcInfo.info }}</span>
</div>
```
