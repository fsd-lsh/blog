### 环境准备
1. 您需要确保安装gokins服务器上拥有mysql5.7
2. 如果使用云服务器确保gokins默认端口8030入口方向开放

### 下载Gokins
1. 服务器下载Gokins可执行程序并移动至usr bin
```
# 进入/usr/bin
sudo cd /usr/bin

# Linux(看具体情况)
wget http://bin.gokins.cn/gokins-linux-amd64 

# Mac(看具体情况)
wget http://bin.gokins.cn/gokins-darwin-amd64

# 修改脚本权限及重命名
chmod 777 gokins-linux-amd64 && mv gokins-linux-amd64 gokins （Mac情况下注意脚本名）

# 运行
./gokins

```

### 进入安装界面
访问 http://你的域名或IP:8030进入到Gokins安装页面 (后续根据提示完成即可)
![e484d9747dec43108325c22283abe39f.png][1]

### gokins想要让其在系统后台运行输入如下命令
```
nohup ./gokins >/dev/null  2>&1 &
```
