**假设有数据表：**

```
CREATE TABLE `order` (
  `order_id` varchar(20) NOT NULL COMMENT '订单号',
  `c_date` timestamp NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建日期',
  PRIMARY KEY (`order_id`) USING BTREE
) ENGINE=MyISAM DEFAULT CHARSET=utf8 COMMENT='订单表';
```

查询后字段c_date直接显示为`2021-03-01 00:00:00`格式，
想要c_date仅查询到年份，需要按照如下方式查询

```
SELECT
    order_id,
    year(createtime) as c_year
FROM
    `order` 
```

这样就可以得到date类型字段查询仅显示年份了
