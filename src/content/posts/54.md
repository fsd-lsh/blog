最近考虑到项目安全，需要限制用户私自修改sessionStorage本地存储。

查阅资料后简单来说就是需要在main.js中添加代码如下
```
//防止手动修改本地存储
window.addEventListener('storage', (e) => {
    sessionStorage.setItem(e.key, e.oldValue);
});
```

添加后打开控制台手动测试一下，确实修改后无效果 SUCCESS！。
