1. pecl下载扩展压缩包
https://pecl.php.net/package/ssh2
查看合适版本的扩展，我这里php使用7.4所以直接下载最新的就可以

2. 解压并编译安装
```
cd ~/desktop
wget https://pecl.php.net/get/ssh2-1.3.1.tgz
tar -zcvf ssh2-1.3.1.tgz && cd ssh2-1.3.1/ssh2-1.3.1
phpize
./configure
make && make install
```
3. php.ini中添加扩展
```
vim /usr/local/etc/php/7.4/php.ini
```
找个合适位置添加`extension="ssh2.so"`

4. 查看扩展是否生效
```
php -m
```
里面有ssh2就OK了
