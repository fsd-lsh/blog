需求需要实现例如将 `23012` 显示为 `23,012`

1.创建转换方法`addThousand`
```vue
methods: {
  addThousand: num => num.toLocaleString(),
}
```

2.在Dom上显示
```vue
<template>
  <h1>{{addThousand(23012)}}</h1>
</template>
```

3.看效果已显示为`23,012`
