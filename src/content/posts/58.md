```
        const url = 'fileApi/AccountTemplate.xlsx'
        const link = document.createElement('a')
        link.style.display = 'none'
        link.href = url
        link.setAttribute(
          'download',
          "导入学生账号模板"
        )
        document.body.appendChild(link)
        link.click()
```

就是body创建一个a标签，设置属性然后 click就行了 上面的"导入学生账号模板"就是自己想修改的下载时文件名
