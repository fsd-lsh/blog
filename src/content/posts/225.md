Taro框架开发微信小程序编译后报错routeDone with a webviewId 18 that is not the current page(env: macOS,mp,1.06.2401020; lib: 3.2.5)

![WX20240212-174241@2x.png][1]

解决办法：在app.config.js增加`lazyCodeLoading : "requiredComponents"`，如下配置
```javascript
export default defineAppConfig({
    pages: [
        'pages/index/index',
    ],
    window: {
        navigationBarTitleText: 'xxx',
    },
    tabBar: {
        list: [
            {
                pagePath: "pages/post/index",
            },
        ],
        custom: true,
    },
    lazyCodeLoading : "requiredComponents"
})

```
